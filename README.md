# 🚀 开源阅读(Legado)web服务更快翻页
> **通过预加载与本地缓存，加快开源阅读(Legado)Web服务的翻页速度。**

![Version](https://img.shields.io/badge/version-2.0.0-blue) ![License](https://img.shields.io/badge/license-GPL--3.0-green)

## 📖 简介

**开源阅读 (Legado)** 是一款功能强大的安卓阅读APP。它的 "Web服务" 功能允许用户在局域网内的电脑或其它设备上阅读手机里的书籍。

然而，原生的 Web 端存在一个体验痛点：**每次翻页都需要发起实时网络请求**。这导致了明显的加载延迟。



## ✨ 核心功能

**该脚本** 通过**预加载**，尝试加快翻页速度。


## 📥 安装方法

1.  **安装管理器**：
    *   推荐使用 [脚本猫 (ScriptCat)](https://scriptcat.org/)
    *   或者使用 [Tampermonkey (油猴)](https://www.tampermonkey.net/)。
2.  **安装脚本**：
    *   [从 Greasy Fork 安装](https://greasyfork.org/zh-CN/scripts/556654)
    *   或者 [从 GitHub 安装](https://raw.githubusercontent.com/caffetsong/legado-fast-page/main/legado-fast-page.user.js)
## 🎮 使用说明


### 键盘操作
*   **下一章**：按 `→` (右方向键)
*   **上一章**：按 `←` (左方向键)
